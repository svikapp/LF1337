
# Svik-server

#### Visit our website

http://svik.netlify.app





## Setting up

Clone the repo

```bash
  git clone https://github.com/svikapp/svik-server
  cd svik-server
```

Install Packages

```bash
    npm install
```

Run the server

```bash
    npm run dev
```
